# FMP Release 1.3.8 16th February 2022

This is a flood map data and application release. The application release updates the Flood Information
Service Links referenced in the start page and flood zones page.

# Instructions

- Execute job FMP_EA_102_UPSERT_DATA (takes 3-4 hours approx)
 using parameters:
	- v202202
	- AB
	- DEF
	- FSA
	- FZ2
	- FZ3

- Execute job FMP_EA_103_COPY_TILE_CACHE (takes 1 hour approx)

Then full code build and auto test run to update the service to version 1.3.8.
